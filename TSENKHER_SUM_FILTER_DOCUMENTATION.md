# Цэнхэр сум шүүлтүүрийн баримтжуулалт (Tsenkher Sum Filter Documentation)

## Зорилго (Goal)
Малчин бааз (malchincamp.mn) вэбсайтын хайлт болон жагсаалтыг зөвхөн "Архангай аймгийн Цэнхэр сум"-ын баазуудаар хязгаарлах.

## Өөрчлөлтүүд (Changes Made)

### 1. Нүүр хуудасны хайлт (Home Page Search)
- **Файл**: `components/search/SearchSection.tsx`
- **Өөрчлөлт**:
    - Анхны утгыг (Default state) "Архангай" аймаг болон "Цэнхэр" сум болгож тохируулсан.
    - Хайлтын хэсгийн тайлбар текстийг "Одоогоор зөвхөн Архангай аймгийн Цэнхэр сумын баазууд нээлттэй байна" болгож өөрчилсөн.
    - Аймаг болон сумын сонголтыг арилгах (Clear) боломжийг хааж, үргэлж Цэнхэр сумыг сонгосон байхаар болгосон.
    - Сумын сонголтыг зөвхөн "Цэнхэр" сум харуулдаг болгож хязгаарласан.

### 2. Жагсаалтын хуудас (Listings Page)
- **Файл**: `app/listings/page.tsx`
- **Өөрчлөлт**:
    - `filteredCamps` логикт "Архангай" болон "Цэнхэр" гэсэн үг ороогүй баазуудыг харуулдаггүй болгож хатуу шүүлт (Mandatory filter) нэмсэн.
    - Хуудасны дээд хэсэгт "Одоогоор зөвхөн Архангай аймгийн Цэнхэр сумын баазууд нээлттэй байна" гэсэн мэдэгдэл нэмсэн.
    - Сум сонгох хэсгээс "Бүх сумд" гэсэн сонголтыг хассан.

### 3. Баазууд хуудас (Camps Page)
- **Файл**: `app/camps/page.tsx`
- **Өөрчлөлт**:
    - `app/listings/page.tsx`-тэй ижил өөрчлөлтүүдийг хийсэн (Хатуу шүүлт, Мэдэгдэл, Default утгууд).

### 4. Нүүр хуудасны онцлох баазууд (Home Page Featured Camps)
- **Файл**: `app/page.tsx`
- **Өөрчлөлт**:
    - Нүүр хуудсан дээр харагдах 8 баазыг зөвхөн Цэнхэр сумынх байхаар шүүж харуулдаг болгосон.

## Буцаан хэвийн болгох заавар (How to Revert)
Хэрэв ирээдүйд бүх аймгийн баазуудыг дахин нээх шаардлага гарвал дараах алхмуудыг хийнэ:

1. `components/search/SearchSection.tsx`:
   - `useState("Архангай")` болон `useState("Цэнхэр")`-ийг `useState("")` болгох.
   - `handleProvinceClear` болон `handleDistrictClear` функцуудыг хэвийн болгох.
   - `districts` шүүлтийг (`.filter(...)`) хасах.
   - Hero текстийг хэвийн болгох.

2. `app/listings/page.tsx` & `app/camps/page.tsx`:
   - `filteredCamps` доторх "Mandatory filter" хэсгийг хасах.
   - `availableDistricts.filter(...)` хэсгийг хасах.
   - UI дээрх мэдэгдлийн (Alert) хэсгийг хасах.
   - `SelectItem` "Бүх сумд"-г буцааж нэмэх.

3. `app/page.tsx`:
   - `filteredYurts` шүүлтийг хасаж, `yurts` хувьсагчийг шууд ашиглах.
